## 2. IMAGE SPECIFICATIONS BY CARD

### Key Standards
- **Aspect Ratio:** 16:9 (all cards use consistent aspect ratio)
- **Recommended Primary Dimensions:** 1200px × 675px (highest quality for scaling)
- **Recommended Card Display Size:** 400px × 225px (in-page render)
- **File Formats:** PNG (preferred for quality) or WebP (for modern browsers)
- **Optimization:** Compressed for web (50-200KB per image)

## 14. FILE OPTIMIZATION GUIDELINES

### PNG Optimization (Recommended Format)
```bash
# Using ImageMagick (command line)
convert source.png -quality 85 -strip optimized.png

# Using online tool: https://tinypng.com
# Upload PNG → Download compressed version (50-70% smaller)
```

### WebP Alternative Format
```bash
# Convert PNG to WebP for modern browsers
cwebp -q 80 source.png -o source.webp

# Result: ~50% smaller than PNG, better compression
```
